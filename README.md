# HVDC-Transmission-System-Simulation-Using-VSC-in-MATLAB-Simulink
🚀 Project Description
This project focuses on the modeling and simulation of a High Voltage Direct Current (HVDC) power transmission system using Voltage Source Converter (VSC) technology in MATLAB and Simulink.
The objective of the project is to analyze the transmission of electrical power between two AC power networks through a DC transmission link, while enabling precise control over power flow and direction.
The developed model consists of two converter stations based on VSC technology, connected through a DC link. Each converter performs bidirectional power conversion between AC and DC, allowing the system to transfer energy efficiently between the two grids.
A control strategy based on abc–dq coordinate transformation was implemented to simplify the control of three-phase quantities. PID controllers were used to regulate the DC voltage, current, and power flow, ensuring stable system performance under different operating conditions.
One of the key features demonstrated in this project is the bidirectional power flow capability. The system allows power transfer to be reversed, meaning that the sending station can become the receiving station and vice versa, while maintaining overall system stability. This capability represents a major advantage of VSC-based HVDC systems, widely used in modern power transmission networks and grid interconnections.
